## How to run
**1.** Clone this project
```
git clone git@github.com/fernandotag/todo-list
``` 
**2.** Open the project folder todo-list in your system's shell
```
cd todo-list
``` 
**3.** install the project's dependencies
```
npm install
```
**4.** run the app
```
npm run dev
```
